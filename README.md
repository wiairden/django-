# 毕设做的Django任务平台网页  


## 使用教程:  

### 1.安装django与python环境  

1.1安装 Python  

访问 https://www.python.org/downloads/ 下载最新版本的 Python 并安装。
在安装过程中，勾选“Add Python x.x to PATH”选项，让 Python 在命令行中全局可用。  

1.2安装 Django
打开命令行（Windows 中使用 PowerShell 或 cmd，Mac 和 Linux 使用终端）。
使用 pip 工具安装 Django。运行以下命令：
>pip install django
